# Portafolio de Estadística Aplicada

Portafolio digital showcasing academic work from the Applied Statistics course at Universidad Central del Ecuador.

## 📁 Estructura del Proyecto

```
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # JavaScript
├── LICENSE             # Licencia MIT
└── README.md           # Este archivo
```

## 🚀 Uso

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. O usa un servidor local:
   ```bash
   npx serve .
   ```

## 📚 Contenido

- **Estadística Descriptiva**: Medidas de tendencia central y dispersión
- **Probabilidades**: Distribuciones binomial y normal
- **Inferencia Estadística**: Pruebas de hipótesis
- **Regresión Lineal**: Análisis de correlación
- **ANOVA**: Análisis de varianza
